# Zuvo Marketplace

Plugin marketplace for [Zuvo](https://github.com/greglas75/zuvo) — auto-activating, multi-agent skill ecosystem for Claude Code.

## Install

```bash
# 1. Add this marketplace (one-time)
claude plugin marketplace add greglas75/zuvo-marketplace

# 2. Install Zuvo
claude plugin install zuvo
```

## What you get

33 skills, 12 specialized agents, quality gates, and structured workflows — all auto-activating.

See [greglas75/zuvo](https://github.com/greglas75/zuvo) for full documentation.
